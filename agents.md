# Repository Working Memory

## Review Persistence

- Every review or re-review must be saved as a new document.
- Never overwrite or append into a prior review document when the user asks for a fresh review round.
- Default location for saved review documents is `docs/` unless the user specifies another path.
- When a target already has prior review rounds, create a new filename for the new round so the review history remains traceable.

## Current Naming Guidance

- For reviews of `docs/enhancement-plan-2026-04-12.md` phase 10, keep using separate round-specific files under `docs/`.
